Aufgaben

Achtung: Teste dein Wissen mit den folgenden Assignments.

Summe von Zahlen

Github Repository

  1. Erstelle eine `sum_numbers.py` Datei im Root-Verzeichnis Deines Repositories.
  2. Schreibe eine Funktion `sum_numbers`, die eine Liste von Zahlen als Argument akzeptiert und die Summe dieser Zahlen zurückgibt.
Beispiel: 
sum_numbers([1, 2, 3, 4])  # Sollte 10 zurückgeben  
sum_numbers([])  # Sollte 0 zurückgeben

Flächeninhalt eines Kreises

Github Repository

  1. Erstelle eine `circle_area.py` Datei im Root-Verzeichnis Deines Repositories.
  2. Schreibe eine Funktion `circle_area`, die den Radius als Argument akzeptiert und den Flächeninhalt des Kreises zurückgibt. Verwende die Formel A = π * r^2
Beispiel: 
circle_area(3)  # Sollte 28.27 zurückgeben  
circle_area(0)  # Sollte 0 zurückgeben

Zeichenlänge eines Strings

Github Repository

  1. Erstelle eine `string_length.py` Datei im Root-Verzeichnis Deines Repositories.
  2. Schreibe eine Funktion `string_length`, die einen String als Argument akzeptiert und die Anzahl der Zeichen zurückgibt.
Beispiel: 
circle_area(3)  # Sollte 28.27 zurückgeben  
circle_area(0)  # Sollte 0 zurückgeben
